Etiquetas estructurales
Etiqueta Descripcion Propiedades Ejemplo
< input > Ingreso texto Se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario. < input type="text" >
< menu> Menu / contenido Representa una lista desordenada de elementos id < div class="news">
< a href="#">NASA’s Webb Delivers Deepest Infrared Image of Universe Yet< /a>
< menu>
< li>< button id="save">Save for later< /button>< /li>
< li>< button id="share">Share this news< /button>< /li>
< /menu> < /div>
< nav > Link de navegacion Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, < nav class="menu">
< ul>
< li>< a href="#">Inicio< /a>< /li>
< li>< a href="#">Sobre nosotros< /a>< /li>
< li>< a href="#">Contacto< /a>< /li>
< /ul>
< /nav>
< base > Link de navegacion Especifica la dirección URL base que se utilizará para todas las direcciones URL relativas contenidas dentro de un documento < base href="http://www.example.com/page.html" />
< base target="_blank" href="http://www.example.com/page.html" />
< button> Click Representa un elemento cliqueable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite un botón estándar y simple de aplicar. < button name="button">Click me< /button>
< datalist > Lista / Opcion Contiene un conjunto de elementos < option> que representan los valores disponibles para otros controles. < label>Choose a browser from this list: < input list="browsers" name="myBrowser"/>< /label>
< datalist id="browsers">
< option value="Chrome">
< option value="Firefox">< /option>
< option value="Internet Explorer">< /option>
< option value="Opera">< /option>
< option value="Safari">< /option>
< option value="Microsoft Edge">< /option>
< /datalist>
< fieldset > Contenido El elemento fieldset (grupo de campos) permite organizar en grupos los campos de un formulario. < fieldset>
< legend>Información Personal< /legend>
Nombre: < input name='nombre' type='text' tabindex='1'>
Apellidos: < input name='apellidos' type='text' tabindex='2'>
< /fieldset>
< link > Click Especifica la relación entre el documento actual y un recurso externo. Los usos posibles de este elemento incluyen la definición de un marco relacional para navegación. < link href="default.css" rel="stylesheet" title="Default Style" /> < link href="fancy.css" rel="alternate stylesheet" title="Fancy" />
< samp > Contenido Se utiliza para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador. < p> < samp>Keyboard not found
Press F1 to continue< /samp> < /p>
< output > Contenido Es un elemento contenedor en el que un sitio o aplicación puede inyectar los resultados de un cálculo o el resultado de una acción del usuario. < form id="example-form">
< input type="range" id="b" name="b" value="50" /> +
< input type="number" id="a" name="a" value="10" /> =
< output name="result" for="a b">60< /output>
< /form>
< details > Contenido Es usado como un widget de revelación a través del cual el usuario puede obtener información adicional < details>
< summary>Some details< /summary>
< p>More info about the details.< /p>
< /details>
